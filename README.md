# 💰 Suivi Financier Familial

Une page web simple et responsive pour suivre les entrées et dépenses d'une famille, directement à partir d'un fichier Excel.

## 🚀 Fonctionnalités

- 📊 Affichage des données sous forme de tableau
- 💵 Résumé des finances (solde total, entrées, sorties)
- 🔍 Recherche par désignation ou observation
- 📅 Filtrage par période (date de début et date de fin)
- 📱 Design responsive (mobile, tablette, desktop)
- 🎨 Icônes et design familial

## 🛠️ Utilisation

1. **Placez votre fichier Excel** nommé `donnees.xlsx` dans le même dossier que les fichiers HTML/CSS/JS.
2. **Format du fichier Excel** : les colonnes doivent être dans l'ordre :
   - A : Date
   - B : Désignation
   - C : Entrées
   - D : Sorties
   - E : Solde
   - F : Observations

   Ou bien vous pouvez utiliser des en-têtes nommés : `Date`, `Désignation`, `Entrées`, `Sorties`, `Solde`, `Observations`.

3. Ouvrez `index.html` dans votre navigateur.
